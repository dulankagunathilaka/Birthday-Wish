# Happy Birthday Kavi

A simple animated birthday website for Kavi from Dulanka Nimsara.

## Add the images

Place your five photos in an `assets` folder with these names:

- `image01.jpg`
- `image02.jpg`
- `image03.jpg`
- `image04.jpg`
- `image05.jpg`

If your files use a different extension, update the `src` values in `index.html`.

## Publish on GitHub Pages

1. Create a GitHub repository and push these files.
2. In the repo settings, open Pages.
3. Set the source to the `main` branch and the root folder.
4. Save and wait for the Pages link to appear.

## Open locally

Open `index.html` in a browser, or use GitHub Pages after publishing.
